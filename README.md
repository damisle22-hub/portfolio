# DI/OS — Interactive Portfolio

![License](https://img.shields.io/badge/license-MIT-blue)
![HTML](https://img.shields.io/badge/HTML-5-orange)
![CSS](https://img.shields.io/badge/CSS-3-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)

An interactive operating system-inspired portfolio built entirely with **HTML, CSS, and vanilla JavaScript**.

Instead of scrolling through a traditional portfolio website, visitors boot into **DI/OS**, a fictional desktop operating system where projects, experience, skills, and contact information are presented as draggable desktop applications.

🌐 **Live Demo:** https://damisle22-hub.github.io/portfolio/

---

## Features

### System Experience
- BIOS startup sequence
- Animated operating system boot screen
- User login screen
- Custom operating system sound effects
- Animated desktop environment
- Live system monitor (CPU, RAM, Power)
- Dynamic taskbar clock

### Desktop Environment
- Draggable desktop icons
- Multi-select icons
- Drag-selection box
- Right-click context menus
- Start Menu
- Draggable windows
- Resizable windows
- Minimize / Maximize / Close controls
- Taskbar window management

### File System
- Interactive Trash Bin
- Drag files into the Trash
- Restore deleted items
- Empty Trash functionality
- Multi-select support inside Trash

### Portfolio Content
- About
- Skills
- Experience
- Projects
- Resume
- Contact Information

---

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- Canvas API
- Web Audio API

No frameworks or libraries were used.

---

## Why I Built This

Most portfolios follow the same formula:

- Hero section
- About
- Projects
- Contact

I wanted to build something that people would actually remember.

DI/OS transforms a portfolio into an interactive desktop experience where visitors naturally explore my background the same way they would explore a real operating system.

## Running Locally

Clone the repository:

```bash
git clone https://github.com/damisle22-hub/portfolio.git
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Then open:

```
http://localhost:3000
```
